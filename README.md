# AI Coding Plan 对比平台

一个用于对比国内主流AI编码服务套餐的纯前端应用。

## 访问应用

您可以通过以下链接访问已部署的应用：

[https://koalahao.github.io/CodingPlans-cn/](https://koalahao.github.io/CodingPlans-cn/)

## 功能特点

- **对比功能**：支持6个国内AI平台的编码套餐对比
- **详细信息**：查看每个平台的套餐详情、价格和额度
- **筛选排序**：根据价格、额度等条件进行筛选和排序
- **响应式设计**：适配各种屏幕尺寸
- **纯前端实现**：无需服务器，直接通过浏览器访问

## 支持的平台

- 智谱AI
- MiniMax
- 火山引擎
- Streamlake(快手)
- Kimi
- 阿里云百炼

## 技术栈

- HTML
- CSS
- JavaScript
- Git
