# AI Coding Plan 对比 App 开发计划 (纯前端)

## 技术栈选择
- **纯HTML + CSS + JavaScript** (无需构建工具)
- **UI组件库**: Ant Design (通过CDN引入)
- **样式**: Tailwind CSS (通过CDN引入)
- **图标**: Lucide Icons (通过CDN引入)
- **数据源**: 本地JSON文件或直接嵌入HTML中

## 项目结构
```
aicoding-compare/
├── index.html           # 主页面
├── data.json            # 所有平台数据
├── css/
│   └── styles.css       # 自定义样式
└── js/
    ├── app.js           # 主应用逻辑
    ├── components.js    # 组件逻辑
    └── utils.js         # 工具函数
```

## 核心功能

### 1. 首页展示
- 6个服务商的概览卡片
- 显示关键信息：名称、最低价格、最高额度、特点
- 卡片点击跳转到详情页

### 2. 对比页面
- 详细对比表格（可按套餐对比）
- 支持多选服务商和套餐进行横向对比
- 对比维度：价格、月度额度、5小时限额、支持模型、适配工具、特点

### 3. 详情页面
- 单个服务商的完整信息
- 显示所有套餐详情
- 用户评价和特点
- 官网链接

### 4. 筛选和排序
- 按价格排序（升序/降序）
- 按月度额度排序
- 按服务商筛选
- 按套餐类型筛选
- 价格范围筛选

## 实施步骤
1. 创建 index.html 文件（引入CDN依赖）
2. 创建 data.json 文件
3. 创建 css/styles.css 文件
4. 创建 js/app.js 主应用逻辑
5. 创建 js/components.js 组件逻辑
6. 创建 js/utils.js 工具函数
7. 实现首页展示功能
8. 实现对比表格功能
9. 实现详情页面功能
10. 实现筛选和排序功能
11. 优化响应式设计
12. 测试和调试

## 使用方式
直接双击 index.html 文件即可在浏览器中打开使用，无需任何构建工具或服务器。
